## 查看系统激活信息

### slmgr.vbs

https://jingyan.baidu.com/article/48206aead03c57216bd6b369.html

```
slmgr.vbs -xpr  //显示当前操作系统是否为永久激活。如果不是，则给出截止日期。
slmgr.vbs -dlv  //列出详细的激活信息，含激活ID、安装ID、激活截止日期。
slmgr.vbs -dli   //列出当前操作系统的版本、一部分产品密钥、激活类型以及许可证状态
```

> 命令提示符，管理员下

```
slmgr vbs /upk  //卸载之前的密钥
slmgr /ipk *****-*****....
slmgr /skms 服务器
```


## linux ubuntu 无法调节亮度

https://www.cnblogs.com/xu-neal/p/3940393.html

> sudo vim /etc/default/grub
> 在打开文件中找到

GRUB_CMDLINE_LINUX=””

改成

GRUB_CMDLINE_LINUX=”acpi_backlight=vendor“

改好后保存即可

> sudo update-grub
> 终端输入代码:

sudo vim /etc/rc.local

在打开文件里增加一句（加在exit 0之前）

echo 500 > /sys/class/backlight/intel_backlight/brightness

然后保存即可


## ubuntu 宽带链接

> 安装 pppoecof
> sudo apt-get install pppoeconf

> 配置
> pppoeconf
> 开始配置

最后会显示plugin rp-pppoe.so loaded 说明链接成功。

> 关闭链接
> poff dsl-provider

> 下次开系统
> pon dsl-proveder即可。
